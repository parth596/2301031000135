Should we use NEWS2 to assess unwell COVID-19 patients in primary care?
NEWS2 is an early warning score developed for monitoring hospital patients. There is no data on its value in COVID-19 in a primary care setting. If using NEWS2, do so alongside a full clinical assessment. Further research on early warning scores for COVID-19 is needed.
